Сайт для мебельной фабрики "Prestige", специализирующейся на производстве корпусной мебели на заказ.

Особенности проекта
- адаптивный дизайн 
- Интерактивный каталог с фильтрацией
- Калькулятор стоимости мебели
- Форма онлайн-заказа

Технологии
- HTML5 
- CSS3
- JavaScript
